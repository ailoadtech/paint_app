## Features

- **Free drawing** - with adjustable brush size by clicking the size button multiple times
- **Color picker** — 24-hue color wheel for easy color selection
- **Stamp tool** — place pre‑made stamps
- **Rainbow mode** — animated color‑cycling brush
- **Clear canvas** — double click toggle for background color cycle between black and white
- **Save paintings** — export as JPEG

## Usage

- **Brush**: Draw on the canvas. Use the size button to cycle brush sizes.
- **Color**: Tap the color button to open the 24‑hue color wheel.
- **Stamp**: Tap the stamp button to cycle through stamps. Tap the canvas to place the selected stamp.
- **Rainbow**: Toggle rainbow mode to paint with an animated color spectrum.
- **Clear**: Toggle canvas background between black and white.
- **Save**: Save the current painting as a JPEG file.

## Technical Notes

- Written by AI in Kotlin using AndroidX and standard Canvas drawing.
- Color wheel draws 24 HSV sectors; touch maps directly to the displayed hue.
